# 🎓 School Management System

A modern full-stack **School Management System** built with **Spring Boot** and **Flutter**, designed to simplify school administration through role-based access control, attendance management, grading, scheduling, applications, and event management.

> 🚧 **Project Status:** Under Active Development

---

## ✨ Features

### 👤 Authentication & Authorization

* JWT Authentication
* Role-Based Access Control (RBAC)
* Refresh Token
* Secure REST APIs

### 👨‍🎓 Student

* View profile
* View timetable
* Check attendance
* View grades
* Submit applications
* View school events

### 👨‍🏫 Teacher

* Manage attendance
* View teaching schedule
* Enter and update grades
* Homeroom class management
* Submit applications
* View school events

### 👨‍👩‍👧 Parent

* Monitor student attendance
* View academic results
* Track timetable
* Receive school notifications

### 🛠️ Administrator

* User Management
* Class Management
* Subject Management
* Timetable Management
* Attendance Management
* Grade Management
* Event Management
* Application Approval
* System Configuration

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/justlamvt05/School-Management-System.git
```

### Backend

```bash
cd backend
```

Configure your PostgreSQL database and run:

```bash
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```
